What is HTML..?

HTML - stands for Hyper Text Markup Language.

HTML describes the structure of a web page. It consists of a series of elements and those elements tell the browser how to display the content.

The <!DOCTYPE html> declaration defines that this document is an HTML5 document

The <html> element is the root element of an HTML page

The <head> element contains meta information about the HTML page

The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)

The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

The <h1> to <h6> elements define headings

The <p> element defines a paragraph


HTML Links

HTML links are defined with the <a> tag.

Example: <a href="https://www.w3schools.com" target="_blank">Visit W3Schools</a>

Visit W3Schools

HTML Images

HTML Images are defined with the <img> tag.

Use a <figure> element to mark up a photo in a document, and a <figcaption> element to define a caption for the photo.


        <figure align="middle">
            <img src="./img/Panda.jpg" alt="Panda" width="600px" height="300px">
            <figcaption style="color:blue;">Panda</figcaption>
        </figure>

HTML Attributes:

The HTML Attributes provide additional information about HTML elements.

These are always specified in the start tag. All HTML elements can have attributes.

1. href:

The href attribute of <a> specifies the URL of the page the link goes to.

2. src:

The src attribute of <img> specifies the path to the image to be displayed.

3. width & height:

The width and height attributes of <img> provide size information for images.

4. alt:

The alt attribute of <img> provides an alternate text for an image.

5. style:

The style attribute is used to add styles to an element, such as color, font, size, and more.

6. title:

The title attribute defines some extra information< an element.


HTML Formatting Elements:

Formatting elements were designed to display special types of text:

The <i> defines italic text. eg. Italic

The <b> defines bold text. eg. Bold

The <strong> defines important text. eg. Strong

The <em> defines emphasized text. eg. Emphasize

The <small> defines smaller text. eg. Small

The <mark> defines marked/highlighted text. eg. Marked/Highlighted

The <sup> defines superscripted text. eg. 264

The <sub> defines subscripted text. eg. H2O

The <ins> defines inserted text. eg. Rs.500

The <del> defines deleted text. eg. Rs.1000


HTML Multimedia:

HTML Multimedia can be almost anything you can hear or see, like images, music, sound, videos, records, films, animations, and more.

HTML Audio:

The HTML <audio> element is used to play an audio file on a web page.

The controls attribute adds audio controls, like play, pause, and volume.

Add muted and autoplay to let your audio file start playing automatically but muted.


            <audio controls muted>
                <source src="./audio/Let Me Love You -DJ Snake, Justin Bieber.mp3" type="audio/mpeg">
                Your browser does not support the audio element.
            </audio>

Let Me Love You - DJ Snake, Justin Bieber


HTML Video:

The HTML <video> element is used to play an video file on a web page.

The controls attribute adds video controls, like play, pause, and volume.

Add muted and autoplay to let your video file start playing automatically but muted.


            <video width="70%" height="70%" controls muted>
                <source src="./video/Butterfly - 20704.mp4" type="video/mp4">
                Your browser does not support the video tag.
            </video>


HTML iframe:

The HTML <iframe> tag specifies an inline frame which is used to embed another document within the current HTML document.

The height and width attributes specify the size of the iframe.


            <iframe width="800px" height="500px" src="https://www.youtube.com/embed/EZASl9RXfCw"
                title="YouTube video player" frameborder="0" allow="accelerometer; autoplay; clipboard-write;
                encrypted-media; gyroscope; picture-in-picture" allowfullscreen>
            </iframe>

Iframe - Target for a Link

An <iframe> can be used as the target frame for a link.

The target attribute of the link must refer to the name attribute of the iframe.


        <iframe src="https://www.zoho.com/" name="iframe_a" height="500px" width="900px"title="Iframe Example"></iframe>

        <a href="https://www.zoho.com/" target="iframe_a">Zoho Website</a>
        <a href="https://www.youtube.com/embed/j-DV1KZZmwc" target="iframe_a">Zoho Office Tour</a>
